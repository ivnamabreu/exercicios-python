## Objetivo

Praticar o uso de variáveis e formatação de strings para organizar e apresentar informações. Neste exercício, você irá trabalhar com dados simples e aprender a estruturá-los de forma clara.

## Especificação

### Exibição de Informações Pessoais

Abra o arquivo main.py. Dentro dele, localize a função resposta.

A função deverá receber três valores:

- Nome (string)  
- Idade (número inteiro)  
- Altura (número decimal)  

E deverá retornar uma string formatada com essas informações de forma organizada.

Caso o exercício original peça entrada de dados com `input()`, considere que esses valores serão passados como parâmetros para a função resposta.

Regras:

- Utilizar os valores recebidos para montar a resposta
- Organizar a saída em uma única string
- Manter uma formatação clara e legível

Exemplos:

- `resposta("João", 25, 1.75)` → `"Nome: João | Idade: 25 | Altura: 1.75"`  
- `resposta("Maria", 30, 1.60)` → `"Nome: Maria | Idade: 30 | Altura: 1.60"`  

**Atenção:** utilize return, não print
